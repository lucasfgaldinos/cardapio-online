# Cardápio Digital

Um cardápio online mobile-first que permite que clientes de restaurantes e lanchonetes visualizem produtos e façam pedidos diretamente via WhatsApp, usando apenas um QR Code.

---

## Funcionalidades

- Visualização de categorias e produtos
- Adição de produtos ao carrinho
- Visualização do carrinho com subtotal
- Envio do pedido para o WhatsApp do restaurante
- Layout responsivo, otimizado para dispositivos móveis

---

## Tecnologias

- **Frontend:** React + TailwindCSS
- **Deploy:** Vercel
- **Integração WhatsApp:** link formatado via `wa.me`
